Qué ofrece APDocentePBA

Una forma práctica de seguir publicaciones de APD sin depender de revisar la página manualmente a cada rato.

01

Alertas personalizadas

El sistema puede filtrar por distrito principal, otros distritos, materia o área, cargo de interés, turno y nivel o modalidad.

02

Menos tiempo perdido

En lugar de revisar APD de forma manual muchas veces por día, recibís un aviso cuando aparece una publicación compatible con tu perfil.

03

Base organizada

El sistema registra ofertas detectadas, docentes cargados, envíos realizados y futuras mejoras para preferencias editables.

Cómo funciona

El circuito fue pensado para que el docente reciba un aviso claro y útil lo antes posible.

1

Te registrás

Cargás tu correo electrónico y completás tus preferencias de búsqueda.

2

Se monitorean publicaciones

El sistema revisa publicaciones públicas de APD y detecta nuevas ofertas.

3

Se comparan criterios

La oferta se cruza con distrito, materia, cargo, turno y otras preferencias del docente.

4

Recibís el aviso

Si hay coincidencia, llega una alerta por correo con los datos y el acceso para postularse.

Preguntas frecuentes

Algunas dudas comunes sobre el funcionamiento del sistema y las alertas.

¿Es una web oficial del ABC?

No. APDocentePBA es un sistema independiente que monitorea publicaciones públicas y envía avisos automatizados.

¿Cómo me registro?

Desde el botón de registro. El formulario permite cargar correo y preferencias para filtrar las alertas.

¿Puedo cambiar mis preferencias más adelante?

Sí. El proyecto ya está pensado para permitir cambios de materias, cargos, turnos, distritos y estado de alertas.

¿Las alertas llegan por correo?

Sí. El sistema envía correos a docentes cuando detecta una publicación compatible con sus criterios configurados.
