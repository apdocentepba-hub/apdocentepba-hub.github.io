Qué ofrece APDocentePBA

Una manera más práctica de seguir publicaciones de APD sin depender de revisar manualmente la página durante todo el día.

01

Alertas personalizadas

El sistema puede filtrar por distrito principal, otros distritos de interés, materias o áreas, cargo y turno.

02

Menos revisión manual

En vez de entrar una y otra vez a revisar publicaciones, recibís avisos cuando aparece algo compatible con tu perfil.

03

Base organizada

APDocentePBA está pensado para registrar ofertas detectadas, docentes cargados y estado de los avisos enviados.

Cómo funciona

El flujo fue armado para que el aviso llegue de forma clara, rápida y con datos útiles para el docente.

1

Te registrás

Cargás tu correo y tus preferencias desde el formulario.

2

Se detectan publicaciones

El radar revisa publicaciones públicas y registra nuevas ofertas.

3

Se comparan criterios

La oferta se cruza con distrito, materia, cargo, turno y otras preferencias.

4

Llega la alerta

Si coincide con tu perfil, se envía el aviso por correo electrónico.

Preguntas frecuentes

Algunas dudas comunes sobre APDocentePBA y el sistema de notificaciones.

¿Es una web oficial del ABC?

No. APDocentePBA es un sistema independiente que monitorea publicaciones públicas y genera avisos automatizados.

¿Cómo me registro?

Desde los botones de registro de esta web. El formulario permite cargar correo y preferencias.

¿Puedo cambiar mis preferencias?

Sí, el proyecto ya está pensado para permitir actualización de materias, cargos, turnos, distritos y estado de alertas.

¿Las alertas llegan por correo?

Sí. Cuando el sistema detecta una publicación compatible, puede enviar un correo con los datos relevantes.
